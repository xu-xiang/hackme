# ABOUT DVWA
[DVWA](http://www.dvwa.co.uk/)是一款包含SQL注入、文件上传、XSS、文件包含等常见漏洞的漏洞测试模拟环境。
其代码库地址是：https://github.com/ethicalhack3r/DVWA


# 使用方法
docker build -t dvwa .
docker run --name dvwa -p 80:80 -d dvwa
